  • About
    This site lets you search for (your favourite) Marvel characters, and gets information for them from Marvel's database.
    You will be able to see which comics the characters were in, which series and which events they were a part of.
    This site was built as a project, where the goal was to create a single page application, relying heavily on an API.
    Being me, having grown up on Marvel Comics, the choice for an API to use was quickly decided.
    As it turns out, through the API, access is gained to information related to well over 1400 characters, and all the comics, series and events related to them.
    There are plenty of heroes and villains to pick from, so have a look, see what (and who) you can find.
    Auto-completion will help you find whomever you're looking for.
  • Notes
    Searches can take some time (looking at you Spider-Man!), as all data is retrieved from the Marvel database.
    I have made sure the overall wait is limited however. But do keep in mind, some searches are more extensive. There are also times when the servers at Marvel have an internal faux-pas, which will be shown to you as a disabled result button.
    Should that happen, please re-do the search, usually it's a one-off glitch.
    A good number have a description with them, but not all of them. In all cases, a link is provided to the character's entry on the Marvel pages, should you want to explore them further.
    An option to be created still, is to select the year for which you'd like to have results shown.
    Right now, all results come available, but in some cases that means to get to the oldest comics for instance, very high page numbers need to be selected.
    In the case a search returned empty, or an error occurred on Marvel's side, it will be clear that no content was returned.
    Any content returned will be behind a red button, disabled buttons will be opaque and have a notification text on them.
    Zoom in! Most of the images on the site can be tapped or clicked for a zoomed-in view.
    Whenever an image is part of a group (such as the comics for a results page), you'll have a slideshow.
